# Likeastore Pro site

### Developing

1. Clone this repository

2. Run ``npm install``

3. Run ``gulp server``

4. Check in your browser ``localhost:3003``

### Deploying

1. Run ``gulp``

2. Commit changes.

3. Run ``git subtree push --prefix public production master``
